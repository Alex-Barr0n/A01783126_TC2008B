HTML cheat sheet


a - Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: Este link definitivamente no lleva a un video de Rick Astley.


p - Párrafo

La idea de un párrafo es agrupar componentes de una página para dar más estructura.

Por ejemplo: La oración de arriba esta estructurada como un párrafo utilizando <p>.


h1, h2, ... , h6 - Encabezados

Un encabezado permite segmentar una página web.

Es posible cambiar el tamaño de del encabezado cambiando el número de encabezado.

Por ejemplo:

Encabezado nivel 1 utilizando <h1>

Encabezado nivel 2 utilizando <h2>

Encabezado nivel 3 utilizando <h3>

Encabezado nivel 4 utilizando <h4>

Encabezado nivel 5 utilizando <h5>
Encabezado nivel 6 utilizando <h6>

div - Contenedor de bloque

Una división funciona para seccionar un documento y funciona para poder aplicar estilo con CSS.

Este es el encabezado en el elemento división.

div - Contenedor de línea

Un contenedor de línea funciona para editar o cambiar una parte del texto en cuanto estilo

Por ejemplo: Este es un mismo texto. Y este es un span dentro del texto.


strong - Texto con mayor importancia

Como lo indica el título el tag <strong> destaca un texto con mayor importancia resaltándolo en negritas.

Por ejemplo: CUIDADO este link podría ser un virus... o no.


i - Cursiva

La letra italica se utiliza para indicar un término técnico, otro idioma, un pensamiento, etc.

Por ejemplo: Esta frase esta en español. This phrase is in English.


u - Subrayado

El subrayado representa texto con un estilo diferente al del texto normal.

Por ejemplo: En esta frace hay una palabra mal escrita.


b - Negritas

Al contrario de <strong> las negritas solo resaltan el texto, este texto no tiene una importancia extra.

Por ejemplo: Hay una palabra en negritas en esta frase.


strike - Tachar

Para tachar un texto se utiliza esta función

Por ejemplo: Normal. Tachado


small - Texto pequeño

Se utiliza mayormente para un texto secundario. La letra se hace más pequeña a la que se esta utilizando.

Por ejemplo: Texto normal. Texto secundario.


em - Enfatizar texto

Aunque se muestra en itálicas, pero a diferencia de <i> se usa para enfatizar una parte del texto.

Por ejemplo: Es de vital importancia entregar este documento a tiempo.


header - Encabezado

Funciona como un contenedor de contenido introductorio

Por ejemplo: en esta página el <header> es el texto que dice "HTML cheat sheet" en la parte superior.


section - Sección

Se utiliza para definir una sección en una página.

Por ejemplo: En esta página hay diferentes secciones para los diferentes elementos.
