color

Esta propiedad sirve para darle color al texto.

CSS tiene muchas maneras de establecer un color:

  • rgb(72, 49, 100)
  • #483164
  • hsl(267, 34%, 29%, 1)
  • rgba(72, 49, 100, 1)

Todas estas son maneras diferentes de escribir el mismo color (el que está de fondo para esta seccón).

Sin embargo, CSS/HTML admite 140 nombres de color estándar. Aquí los puedes ver.
